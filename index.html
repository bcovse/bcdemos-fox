    Feature Description Documentation/Links
    Feed Selectors Using AJAX and ReactJS we read the feeds from the Fox servers and populate the two selectors. When a feed and a video has been selected the player source is set to the selected video. Perform Player
    Social Plugin The social plugin is configure for this player so that once the video has been played the user is presented with options to shared the video on his or her social networks. Social Plugin
    Advertising The player has been configured to use the Freewheel plugin to display a canned ad. The ad will be displayed as a pre-roll and also as a mid-roll at 1 minute in to the video. Freewheel Plugin
    Moat Analytics The player has been configured to use the Moat plugin. This is only intended to show how easy it is to setup up the plugin in the Brightcove Player. In a real world scenario you would work with your Moat representative to configure Moat then plugin that configuration in to the player. Moat & Brightcove
    Comscore Analytics The player has been configured to use the Comscore plugin. This is not connected to a real Comscore account but it does fire the events. Comscore Plugin
    Adobe Analytics The player has been configured to use the Omniture plugin. This is not connected to a real Omniture account but it does fire the events. Omniture Plugin
